5 Minutes A Day

5 Minutes A Day is an Android application that gives users daily 5-minute de-stressing activities in categories including art, exercise, and mindfulness. After doing an activity, the user records how relaxed he or she feels through a five-star rating system. The app then tracks which activities users have completed, when they have completed them, and how relaxed the user felt after completing each activity.

Built Using:

  • Android Studio
  • Java
