PetitionFinder

PetitionFinder is an Android application that provides users an easy, personalized way to find petitions that fit with their interests. The application asks users how important different types of issues are to them. Then, it uses their preferences to display petitions surrounding issues they care most about along with short descriptions of each of them.

Built Using:

  • JAVA
  • Android Studio
