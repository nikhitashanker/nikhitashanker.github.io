UC Brainstorm

UC Brainstorm is a web application that helps users brainstorm answers to the UC Application's short answer questions. The application allows user to develop an awards and activities list through a form. It then prompts user to decide which awards and activities are suitable to include in responses to each of the eight UC short-answer questions. The application generates a table with each of the activities and awards along with the question they best answer and an explanation as to why that is the question they best answer.

Built Using:

  • Python
  • HTML
  • CSS
  • Django Framework
  • Bootstrap
